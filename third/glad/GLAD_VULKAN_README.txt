========================================================================
       GLAD2 VULKAN — glad2 配置参考
========================================================================

本目录包含 glad2 生成的 Vulkan 运行时加载器代码。

完整的项目技术文档（包括 glad2 集成架构、Header Hijacking、
兼容层、5大平台策略等）已合并到项目根目录：

    → PROJECT_GUIDE.md

以下仅保留 glad2 目录自身的说明。

------------------------------------------------------------------------
文件清单
------------------------------------------------------------------------
include/glad/vulkan.h       glad2 自动生成的主头文件（不要修改）
include/vulkan/vulkan.h     伪装头文件 → #include <glad/vulkan.h>
include/vulkan/vulkan_core.h 伪装头文件 → #include <glad/vulkan.h> + 兼容层
include/vulkan/vulkan_win32.h 伪装头文件 → #include <glad/vulkan.h>
src/vulkan.c                glad2 自动生成的加载器实现（不要修改）

------------------------------------------------------------------------
重新生成 glad2 代码
------------------------------------------------------------------------
1. 访问 https://gen.glad.sh/
2. 配置：C/C++, Vulkan 1.3, loader + on-demand
3. 扩展列表见 PROJECT_GUIDE.md 第 4.2 节
4. 生成后替换 include/glad/ 和 src/ 下的文件
5. 不要替换 include/vulkan/ 下的伪装头文件（这些是我们手写的）
6. 重新生成后检查 vulkan_core.h 兼容层是否需要更新

------------------------------------------------------------------------
重要约束
------------------------------------------------------------------------
1. 不要修改 glad2 自动生成的代码（vulkan.h / vulkan.c）
2. 如需补充类型或宏，在 include/vulkan/vulkan_core.h 兼容层中添加
3. 不要在 CMake 中链接官方 Vulkan 库
========================================================================
